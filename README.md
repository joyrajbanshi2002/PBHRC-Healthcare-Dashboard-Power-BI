# PBHRC-Healthcare-Dashboard-Power-BI
This project is an interactive Healthcare Analytics Dashboard developed in Power BI using the PBHRC Demo Dataset. The dashboard provides actionable insights into patient appointments, consultation trends, revenue, cancellations, and operational performance, enabling data-driven decision-making for healthcare management.
📊 Dashboard Features
📅 Total Bookings Analysis (5,000 Appointments)
💳 Paid vs Free Consultation Analysis
📈 Total Revenue Tracking
❌ Cancellation Rate Analysis
🚫 No-Show Rate & Arrival Rate Monitoring
👨‍⚕️ Doctor-wise Revenue Analysis
🏥 Clinic-wise Revenue Analysis
🌐 Appointment Source Analysis (Walk-in, Website, Google, Facebook, etc.)
📅 Booking Trend by Appointment Date
📋 Appointment Status Summary
🎯 Interactive Filters (Doctor, Gender, Clinic, Consultation Type)
🛠️ Tools & Technologies

📈 Key Insights
Identified high-performing doctors and clinics based on revenue.
Analyzed cancellation and no-show percentages to improve operational efficiency.
Compared appointment sources to understand patient acquisition channels.
Monitored booking trends and consultation distribution over time.
Built dynamic KPIs for quick business performance evaluation.
📂 Dataset

This dashboard is built using the PBHRC Demo Healthcare Dataset, which contains appointment, consultation, revenue, doctor, clinic, and patient information.
